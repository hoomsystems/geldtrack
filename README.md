# GeldTrack

Sistema de gestión de finanzas personales desarrollado con Streamlit.

## Características

- Registro de gastos
- Categorización de gastos
- Múltiples cuentas
- Análisis de gastos
- Presupuestos por categoría
- Soporte multiusuario
- Interfaz multilenguaje (ES/EN/DE)

## Requisitos

- Python 3.8+
- Streamlit
- SQLite3
